## Setup

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build
```

## Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or connect your GitHub repo to Vercel for automatic deployments.

## Usage

1. Enter the prospect's first name (and optionally last name)
2. Select a persona or compare all three side-by-side
3. Copy the appropriate stage message
4. Paste into LinkedIn

All "NAME" placeholders auto-update based on your input.
